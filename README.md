# ☕ Java OOP Exercises

Repositório com exercícios práticos de Programação Orientada a Objetos em Java.

Este projeto reúne implementações desenvolvidas durante meus estudos em Java, com foco na consolidação dos principais pilares da orientação a objetos.

---

## 📚 Conceitos praticados

- ✅ Classes e Objetos
- ✅ Encapsulamento
- ✅ Construtores
- ✅ Sobrecarga
- ✅ Herança
- ✅ Polimorfismo
- ✅ Composição
- ✅ Enum
- ✅ Manipulação de Datas
- ✅ Listas e Coleções
- ✅ Estruturação em pacotes

---

## 📂 Estrutura do Repositório

Cada pasta representa um exercício independente:
poo-bank-exercise
poo-employee-salary
poo-polimorfism-employee
poo-price-product
taxPayer
StringBuilder
...

Os exercícios seguem estrutura padrão:
src/
├── application/
├── entities/
└── Main.java
